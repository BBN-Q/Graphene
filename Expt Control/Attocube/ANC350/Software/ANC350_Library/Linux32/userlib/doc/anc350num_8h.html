Detailed Description

Control functions for ANC350NUM.

Defines additional functions for controlling the ANC350NUM. All common functions from anc350res.h also apply to the ANC350NUM. The functions are not thread safe!

Enable Position Auto Reset.

Enables or disables the position auto reset for an axis. When enabled, every time the reference marking is hit, the position will be set to zero. When disabled, the reference marking will be ignored. Only applicable for ANC350NUM.

Enable Reference Auto Update.

Enables or disables the reference auto update for an axis. When enabled, every time the reference marking is hit, the reference position will be updated. When disabled, the reference marking will be considered only the first time, later hits will be ignored. Only applicable for ANC350NUM.
